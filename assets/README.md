# Assets

This folder is reserved for future public-safe visual assets.

Allowed future assets:

- Synthetic cockpit mockup
- Workflow diagram
- Knowledge structure screenshot using synthetic data
- Public-safe demo walkthrough

Disallowed assets:

- Private notes
- Personal journal entries
- Private chats
- Sensitive project data
- API keys, tokens, credentials, or `.env` files
- Internal commercial materials
- Client/customer information
- Financial records or trading account information

Any future asset should be reviewed before publication to confirm that it contains only public-safe material.
